# Quick Start Guide

Welcome to the Project Quote Application! This guide will help you get started with the main features in minutes.

## 1. Logging In

1.  Open the application in your web browser.
2.  Enter your email address in the login screen.
3.  Click "Sign In".
4.  Check your email (or console in dev mode) for the OTP code.
5.  Enter the OTP code and click "Proceed".

## 2. Creating a Quote

1.  Navigate to the **Quoting Module** from the dashboard.
2.  **Add Items**:
    *   Use the **Product Catalog** on the left to search for items.
    *   Click on an item to add it to your quote.
    *   Alternatively, ask the **AI Assistant** (floating bubble) to "Add 50 bags of cement".
3.  **Review Quote**:
    *   Check the items in the "Current Quote" panel on the right.
    *   Adjust quantities if needed.
4.  **Save/Submit**:
    *   Click "Save Quote" to save it as a draft.
    *   Click "Submit for Approval" to send it to the controller.

![Quoting Module Placeholder](placeholders/quoting_module.png)
*Screenshot: Quoting Module showing Product Catalog and Current Quote.*

## 3. Approving Invoices (Controller)

1.  Log in as a Controller.
2.  Navigate to **Invoice Approval**.
3.  Review the list of "Pending Approval" invoices.
4.  Select your signature from the dropdown.
5.  Click "Approve" on the invoice row.

![Invoice Approval Placeholder](placeholders/invoice_approval.png)
*Screenshot: Invoice Approval list with signature selection.*

## 4. Viewing My Invoices

1.  Navigate to **My Invoices**.
2.  View the list of invoices you have created.
3.  Use the "Year" and "Month" filters to find specific invoices.
4.  Click "View" to see invoice details or "Email" to send it to the customer.
